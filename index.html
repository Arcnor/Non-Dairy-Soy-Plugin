The Non-Dairy-Soy-Plugin by Ed Venaglia

A Google Closure Template Plugin for IntelliJ IDEA 11.x

Description

A plugin to help with the development of Closure Templates (soy files). This plugin includes support for syntax highlighting, code folding, find usages and go to declaration on parameters and templates, and Structure view. This plugin is still under development, and updates are subject to my available free time.

Install

Browse the IntelliJ plugin repository through your IntelliJ IDE

Features

Template and parameter name rename refactoring across files, within files

Symbol Search to find templates

Go To Declaration for parameters templates across files, within files

Comments recognition

Template Code folding

Structure view

Template and parameter usage highlighting

Error checking

Function name syntax highlighting
